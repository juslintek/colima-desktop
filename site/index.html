Everything colima can do — from a UI.

Native everywhere

SwiftUI on macOS, WinUI 3 on Windows, GTK4 on Linux, Bubble Tea in the terminal. No Electron, zero overhead.

One backend brain

A shared Go daemon (gRPC) drives every frontend identically from a single frozen contract.

Full CLI parity

VM lifecycle, profiles, Docker, Kubernetes (k3s), configuration, networking, runtimes, and AI models.

Turnkey

Auto-installs and auto-updates Colima and its dependencies on first launch (Sparkle on macOS).

Remote + local

Drive a local machine or a remote colima/Lima host over SSH; on Windows, local WSL2/Docker too.

AI workloads

Pull, run, and serve models via colima model with krunkit GPU acceleration.

Four frontends, one core.

Install

macOS

Download the DMG from Releases, or build from source:

brew install xcodegen
xcodegen generate
xcodebuild build -scheme ColimaDesktop \
  -destination 'platform=macOS'

Terminal (TUI)

# start the shared daemon
go run ./daemon/cmd --socket /tmp/colima-desktop.sock &
# run the TUI
cd tui && go run . --socket /tmp/colima-desktop.sock
